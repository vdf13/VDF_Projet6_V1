     SOURCE: http://www.tux-planet.fr/connaitre-la-version-de-linux-installe/       Commande pour connaître la version d'une :
Ubuntu :    cat /etc/lsb-release  |   Debian :    cat /etc/issue    | Redhat, d'une Fedora ou d'une Centos :    cat /etc/redhat-release

APRES INSTALLATION
# APPELER FONCTION CONFIGURATION
Source                          https://www.tecmint.com/install-dhcp-server-in-ubuntu-debian/
Debian Ubuntu  edit the file    /etc/default/isc-dhcp-server        INTERFACES="eth0"
Debian Ubuntu  edit the file    /etc/dhcp/dhcpd.conf               
        option domain-name "tecmint.lan";
        option domain-name-servers ns1.tecmint.lan, ns2.tecmint.lan;
        default-lease-time 3600; 
        max-lease-time 7200;
        authoritative;              
        subnet 192.168.10.0 netmask 255.255.255.0 {
        option routers                  192.168.10.1;
        option subnet-mask              255.255.255.0;
        option domain-search            "tecmint.lan";
        option domain-name-servers      192.168.10.1;
        range   192.168.10.10   192.168.10.100;
        range   192.168.10.110   192.168.10.200;
        }
 
 Source           https://tecadmin.net/configuring-dhcp-server-on-centos-redhat/
  CentOS edit the file      /etc/sysconfig/dhcpd                 DHCPDARGS=eth1
  CentOs edit the file      /etc/dhcp/dhcpd.conf 
        option domain-name "tecadmin.net";
        option domain-name-servers ns1.tecadmin.net, ns2.tecadmin.net;
        default-lease-time 600;
        max-lease-time 7200;
        authoritative;
        log-facility local7;
        subnet 192.168.1.0 netmask 255.255.255.0 {
            option routers                  192.168.1.254;
            option subnet-mask              255.255.255.0;
            option domain-search            "tecadmin.net";
            option domain-name-servers      192.168.1.1;
            option time-offset              -18000;     # Eastern Standard Time
	        range   192.168.1.10   192.168.1.100;
        }                           
  Le fichier config dhcpd.conf est identique quelque soit la version Debian, CentOS. Pour simplifier la fonction configuration.
